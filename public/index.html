Pet Finder

This is an introductory backend project. This page only serves to show that I am capable of serving up an html file via a local source port.

Open up insomnia or postman and try out the following endpoints that I fixed with yet untold skill and ineffable finess.

  • // GET localhost:8080, returns this page.
  • // GET localhost:8080/api, returns hello world.
  • // GET localhost:8080/api/v1/pets, returns all pet data.
  • // GET localhost:8080/api/v1/pets/owner?owner=John, returns the names of dogs that owner "John" owns, currently Rover and Fido.
  • // GET localhost:8080/api/v1/pets/Rover, returns data specific to the dog named Rover.

Behold, a pet finder...
